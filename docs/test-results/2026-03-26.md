## 이후 진행한 데이터 보완 작업

### 법률 데이터 검증 및 수정
AI 검증을 통해 법적 오류 수정:

**civil_lawsuit_process.json (민사소송 절차)**
- 사실조회 vs 금융거래정보제출명령 강제력 구분 명확화
- 은행 회신 기간 1~2주 → 2~4주
- 강제집행 절차 추가 (집행문 부여 → 송달/확정증명 → 채권압류 및 추심명령)
- 재산명시/재산조회 제도 추가 (민사집행법 제61조, 제74조)
- 가압류, 지급명령 제도 추가
- 총 기간 2~3개월 → 3~5개월
- 합의금 "20~50%" → "원금 + 지연이자 + 소송비용"
- 소멸시효 수정 ("한 날로부터 10년")
- 비교표 비용 오류 수정 (송달료 11만원 예납)

**prosecution_strategy.json (형사고소 전략)**
- 형사소송법 제223조 → **제237조** (전체)
- 사기죄 공소시효 7년 → **10년**
- 2025.12.23 형법 개정 반영 (20년 이하 징역/5천만원 이하 벌금)
- 구속영장 → **체포영장**
- 합의서 공증 "필수" → **"권장"**
- 공갈죄 안전선 50% → **삭제** (존재하지 않는 기준)
- 반의사불벌죄 아님 안내, 불기소 항고 절차, 형사 조정 제도 추가
- 수사 기간, 환불 확률 등 근거 없는 수치 보수적으로 수정

**laws.json (법률 근거)**
- 사기죄 법정형 2025.12.23 개정 반영
- 공소시효 10년 추가, 처벌 수준 현실화

**procedures.json (경찰 신고 절차)**
- 중복 내용 삭제 → 신고 방법 비교표 + 무료 법률 상담만 유지
- 관할, 온라인 신고 표현 수정
