### 동의어 확장
- **문제**: "소송 포기" → "소취하" 연결 실패 (Reranker 영어 모델 한계)

- **해결**:
  - `chunking.py`: `load_practical_insights()`에서 `related_keywords`를 page_content에 포함 (1줄 추가)
  - `practical_insights.json`: 8개 항목 모두 구어체 동의어 추가
    - 예: "포기, 그만두기, 취소" → pi_08 소취하
    - 예: "계좌 주인 찾기, 계좌 추적" → pi_04 금융거래정보
    - 예: "애가 사기, 청소년 사기" → pi_03 미성년자
