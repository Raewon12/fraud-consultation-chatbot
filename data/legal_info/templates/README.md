# 고소장 템플릿 사용 가이드

## 템플릿 종류 및 용도

### 1. accusation_template.txt (간단 템플릿)
- **용도**: 일반적인 고소장 (경찰서 방문용)
- **특징**: 
  - 간단한 형식
  - 기본 정보만 포함
  - 빠른 작성 가능
- **사용 시점**: 경찰서 직접 방문해서 고소할 때

### 2. accusation_registered_mail.txt (상세 템플릿)  
- **용도**: 등기우편 발송용 고소장
- **특징**:
  - 매우 상세한 형식
  - 육하원칙 철저 적용
  - 작성 안내 포함
  - 3부 작성 안내
- **사용 시점**: 
  - 경찰서에서 소액이라 거부당한 경우
  - 등기우편으로 확실히 접수시키고 싶은 경우
  - 혼자서 완벽한 고소장을 작성해야 하는 경우

## 선택 기준

| 상황 | 추천 템플릿 |
|------|------------|
| 50만원 이상, 경찰서 방문 | accusation_template.txt |
| 소액 사기, 경찰서 거부 예상 | accusation_registered_mail.txt |
| 확실한 증거, 등기우편 선호 | accusation_registered_mail.txt |
| 빠른 처리 원함 | accusation_template.txt |
| 혼자서 완벽하게 작성 필요 | accusation_registered_mail.txt |

## 작성 후 확인사항

- [ ] 모든 빈칸 작성 완료
- [ ] 날짜, 시간 정확히 기재
- [ ] 계좌번호, 전화번호 정확히 기재  
- [ ] 사실만 기재 (감정 표현 제거)
- [ ] 모든 페이지에 서명 또는 날인
- [ ] 증거 목록과 실제 증거 일치 확인
