# 고소장 템플릿 사용 가이드

## 템플릿

### accusation_registered_mail.txt (등기우편 고소장)
- **용도**: 등기우편 발송용 고소장 (중고거래 사기 전용)
- **특징**:
  - 빈칸 채우기 형식으로 일반인도 작성 가능
  - 육하원칙에 따른 시간순 범죄사실 서술 구조
  - 작성 안내 + 등기우편 발송 체크리스트 포함
- **사용 시점**:
  - 경찰서에서 소액이라 거부당한 경우
  - 등기우편으로 확실히 접수시키고 싶은 경우
  - 변호사 없이 혼자 고소장을 작성해야 하는 경우

## 작성 후 확인사항

- [ ] 모든 빈칸 작성 완료
- [ ] 날짜, 시간 정확히 기재
- [ ] 계좌번호, 전화번호 정확히 기재
- [ ] 사실만 기재 (감정 표현 제거)
- [ ] 모든 페이지에 서명 또는 날인
- [ ] 증거 목록과 실제 증거 일치 확인

## 주의사항

- **무고죄 주의**: 허위 사실 기재 시 형법 제156조로 처벌
- **감정 배제**: 욕설이나 감정적 표현은 절대 금지
- **추측 금지**: 확실한 사실만 기재
- **접수 후**: 경찰서에서 출석 요구가 올 수 있으며, 이는 정상적인 절차
